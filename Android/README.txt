How to get Eclipse + ADT:
1) You will need Eclipse + Android Developer Tools (ADT). Get it here https://developer.android.com/sdk/index.html
2) Download Eclipse + ADT, Unzip, run eclipse.exe.
3) Go to Window -> Android SDK Manager. Update to latest.



How to setup:
1) In Eclipse, go to File -> import.
2) Select Android -> existing android code into workspace. Click next.
3) Root Directory is <path>/CRIMP/Android
4) You should see 2 project (appcompat_v7 and CRIMP). Checkbox both of them and click Finish.
5) The app shld now be ready to run.



How to run:
1) Plug in your android device and click run!
